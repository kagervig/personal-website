💡 Heap (Priority Queue) Masterclass

Core Operations:
  • Push (Insert): Add to the end of the array, then Bubble Up until the heap property is restored. Time: O(log N).
  • Pop (Extract): Remove the root (min/max), move the last element to the root, then Bubble Down until the heap property is restored. Time: O(log N).
  • Peek: Return the root element without removing it. Time: O(1).
  • Heapify: Converting an existing array into a heap. Bottom-up approach is more efficient. Time: O(N).
When to use a Heap?

Use a Heap (PriorityQueue in Java) when you need to repeatedly find the K-th largest/smallest element, or when dealing with a Top-K requirement. It is the gold standard for "streaming data" where you don't have the whole set at once.
